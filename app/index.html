Дренаж без щебня

Система Софтрок для отвода воды состоит из перфорированной гибкой трубы, которая окружена изготовленным из пенополистирола наполнителем. По сравнению с дренажами на основе обычного щебня, подобная конструкция позволяет увеличить эффективность на 30–50 %. Трехметровые блоки могут иметь трубы с различными внутренними диаметрами (110, 160, 200 мм).
